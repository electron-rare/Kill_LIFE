# Rituel — Steering (cabinet) (hebdo, 45–60 min)

## Objectif
Valider décisions structurantes, arbitrer options, ajuster roadmap.

## Entrées
- ADR en attente
- Risques majeurs
- Roadmap / jalons

## Agenda
- Décisions (ADR) : accepter/refuser
- Risques : mitigations
- Roadmap : ajustements

## Sortie
- ADR acceptés
- Roadmap mise à jour
- Priorités clarifiées

# Offre Starter — Copilote Opérateur IA

## L'Electron Rare — Factory 4.0

### Le problème
Vos opérateurs perdent du temps à chercher dans les manuels, interpréter des alarmes, et rédiger des rapports. Les données restent cloisonnées.

### La solution
Un **chatbot opérateur intelligent** déployé **sur site** (on-premise). Aucune donnée ne quitte votre réseau.

### Inclus
| Livrable | Détail |
|----------|--------|
| Chatbot opérateur | Interface web, bilingue FR/EN |
| Base documentaire | Ingestion manuels machines (PDF → RAG) |
| Modèle IA local | Mistral/devstral sur serveur standard |
| API compatible | Intégrable dans tout outil existant (Ollama API) |
| Formation | 1j opérateurs + 1j admin |

### Stack
Mascarade Router → Ollama (LLM local) → Qdrant (RAG) → Fake Ollama API
100% open source, pas de licence récurrente.

### Souveraineté données
- Aucune donnée ne sort du réseau local (RGPD natif)
- Modèles exécutés en local sur votre matériel

### Prérequis
- 1 serveur Linux (16 GB RAM, GPU optionnel)
- Réseau local
